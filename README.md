# 🦜 鹦鹉小百科

嗨！这是一个关于绿颊锥尾鹦鹉（Green-cheeked Conure）的照料知识库。

如果你也养了一只绿颊，那你一定知道——它们挑食、贪玩、对世界充满好奇，有时候也会让人揪心。这个仓库就是把散落在各处的鹦鹉知识整理到一起，方便查、方便记、方便分享。

## 怎么用

用 [Logseq](https://logseq.com) 打开这个文件夹就行，首页自动加载分类目录（`pages/开始.md`），左侧栏可搜索关键词或按板块浏览。

## 这知识哪儿来的

内容是 AI 辅助整理的，底子基于公开的禽类营养学和兽医学资料，经过交叉核对。但鹦鹉真的生病了还是要去看兽医，这里的内容旨在科普和参考，不能替代专业诊疗。
