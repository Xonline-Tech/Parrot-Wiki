# 🦜 鹦鹉小百科

嗨！这是一个关于绿颊锥尾鹦鹉的照料知识库。

如果你也养了一只绿颊，那你一定知道——它们挑食、贪玩、对世界充满好奇，有时候也会让人揪心。这个仓库就是想把散落在各处的鹦鹉知识整理到一起，方便查、方便记、方便分享。

## 里面有什么

目前主要攒了这些内容：

- 🍎 **吃什么好，什么不能吃**  
  从苹果、胡萝卜到牛油果、巧克力——哪些可以天天喂，哪些碰都不能碰，一目了然。

- 🔬 **食物里有哪些营养**  
  比如维生素 C 有什么用、缺了会怎样、吃哪些东西能补。

- 🏥 **常见疾病和中毒**  
  腹泻怎么办、特氟龙锅有多危险、啃了含铅的笼子会有什么后果——提前知道，出事不慌。

- 🧠 **行为、身体构造、换羽繁殖**（还在慢慢补充中）

每一个知识点之间都有"超链接"，比如点开苹果会看到它和维生素 C 的关系，点开腹泻会看到哪些食物可能引起它——不孤立，有联系。

## 这知识哪儿来的

内容是 AI 助手帮忙整理的，底子是网上能找到的公开数据，经过交叉核对。但不管怎么说，鹦鹉真的生病了还是要去看医生，这里的知识只是帮你多了解一些，不是替代兽医。
